### 首页快速开始区说明

首页新增"如何快速开始聊天？"区域，包含三步流程：

1. **配置 API** — 选择 DeepSeek、OpenAI-compatible 或其他 Provider，填入 API Key。未配置 API 时仍可使用本地预览模式。
2. **创建角色** — 在创作工坊创建角色卡，设置身份、性格、说话风格和基础规则。
3. **开始聊天** — 进入聊天室，选择角色并创建会话后即可开始角色扮演。

每一步都有可点击按钮跳转到对应页面。

同时新增"需要帮助？"区域，提供数据备份与恢复、API 配置说明、GitHub、用户文档等入口。

### 移动端聊天入口调整说明

移动端顶部栏左上角从单图标改为明显的"会话 ▼"按钮，包含：
- Drama 图标
- "会话"文字
- ChevronDown 箭头

点击行为不变，继续触发会话列表面板。

### 空状态调整说明

移动端聊天页空状态从简单的"开始聊天，请先创建会话"改为更明确的新手引导：

- 标题：开始你的第一段对话
- 说明：先创建一个会话，选择角色后即可开始聊天
- 主按钮：+ 新建会话
- 次按钮：去创作工坊创建角色

同时在移动端会话列表面板底部新增"新建会话"和"去创作工坊创建角色"按钮。
