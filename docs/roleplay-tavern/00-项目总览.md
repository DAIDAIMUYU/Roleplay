# 项目总览

Roleplay Tavern 是一个本地优先的 AI 角色酒馆，用于创建角色、管理世界书与记忆，并进行长会话角色扮演。

## 当前定位

- 本地优先：不登录也能在当前浏览器使用。
- 可选云端同步：登录后可以手动同步角色、会话、世界书、记忆等数据。
- BYOK：用户使用自己的 Provider API Key。
- 安全边界清楚：API Key 不进入导出备份、context runs、日志或公开调试信息。

## 当前核心能力

- 角色卡、提示词模板、世界书、记忆库。
- 聊天房间、消息版本切换、长会话分页。
- Context Builder、上下文控制台、Token 用量与 DeepSeek 缓存诊断。
- 数据导入导出、回收站、本地/云端同步入口。
- `session_only`、`local_device`、`hosted_encrypted` 三种 API 凭据模式。
- 移动端底部导航、PWA manifest、Service Worker、更新提示和离线提示。
