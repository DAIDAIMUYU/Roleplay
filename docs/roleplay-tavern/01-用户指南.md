# 用户指南

这份指南面向普通用户，解释如何从零开始使用 Roleplay Tavern。

## 1. 这是什么

Roleplay Tavern 是一个 AI 角色聊天与创作工具。你可以创建角色、设置世界书和记忆，然后进入聊天房间与角色对话。

不登录也能使用本地模式。登录只是为了开启云端同步和托管加密 API 凭据。

## 2. 三分钟快速开始

1. 打开设置中心，配置 API。
2. 打开创作工坊，创建第一个角色。
3. 打开聊天房间，新建会话并选择角色。

未配置 API 时，可以先熟悉本地创建流程，但不会获得真实 AI 回复。

## 3. 配置 API

API 配置有三种保存方式：

| 模式 | 含义 | 是否跨设备 |
| --- | --- | --- |
| `session_only` | API Key 只在当前页面会话中临时有效，刷新或关闭后可能需要重新填写。 | 否 |
| `local_device` | API Key 只保存在当前浏览器本地，清除网站数据或换设备后需要重新配置。 | 否 |
| `hosted_encrypted` | 登录后 API Key 通过服务端加密保存，前端不展示明文，适合多设备使用。 | 是 |

测试连接只表示当前配置可用，不代表已经启用。需要点击“设为启用”后，聊天才会使用该配置。

## 4. 创建角色

建议至少填写：

- 角色名称
- 身份定位
- 性格特质
- 说话风格
- 开场白

角色卡越清晰，AI 越容易稳定扮演。

## 5. 新建会话

进入聊天房间后，点击“新建会话”，选择一个角色开始聊天。没有角色时，可以先去创作工坊创建角色，也可以创建空白会话。

## 6. 世界书与记忆

- 世界书适合放世界观、地点、组织、人物关系等背景设定。
- 记忆适合放长期事实、关系进展和用户偏好。
- AI 提炼出的候选记忆不会自动启用，需要用户确认。

## 7. 用量与费用

上下文控制台会显示本次请求的基础 token 用量。DeepSeek 还支持缓存命中、费用估算和余额查询。费用仅估算，实际扣费以 Provider 官方账单为准。

## 8. 上下文压缩

长会话接近上下文窗口上限时，可以使用摘要/压缩功能减少早期聊天占用。压缩不会删除原始消息。

## 9. 本地模式与云端同步

- 本地模式：数据保存在当前浏览器。
- 云端同步：登录后可以手动上传或下载数据。
- local_device API Key 不跨设备。
- hosted_encrypted 是云端加密凭据，可跨设备使用。

清除浏览器网站数据、使用无痕模式、换浏览器或换设备，可能导致本地数据无法恢复。建议定期导出备份。

## 10. 移动端使用

手机浏览器访问仍属于网页模式。底部导航固定在屏幕底部，可以切换首页、聊天、工坊和设置。

页面内 PWA 安装提示卡片已下线。如果需要添加到主屏幕，请使用浏览器菜单或系统分享菜单。

## 11. 常见问题

### 为什么没有 AI 回复？

通常是还没有启用 API 配置。请先配置 API、测试连接，并设为启用。

### 为什么手机看不到电脑配置的 API？

电脑端如果使用 `local_device`，配置只保存在电脑浏览器本地。需要跨设备时请使用 `hosted_encrypted`。

### 数据会不会丢？

本地数据依赖当前浏览器站点数据。建议定期导出备份，或登录后开启云端同步。

## 12. 故障排查

- 页面没更新：清理 PWA 缓存或浏览器站点数据。
- API 测试失败：检查 Key、Base URL、模型名、余额和网络。
- 托管凭据没显示：确认登录同一个账号，并刷新设置页 API 配置。
- 移动端按钮被挡：确认使用最新版本，必要时清理旧 PWA 缓存。
