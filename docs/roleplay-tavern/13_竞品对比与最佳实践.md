# 13_竞品对比与最佳实践

## Character.AI

可借鉴：

- 角色发现与公开展示
- 一眼能理解的聊天入口
- 移动端沉浸式聊天
- 角色创建降低门槛

不要照搬：

- 早期不要做大社区
- 不要把公开角色市场作为主线

## Replika

可借鉴：

- 长期关系感
- 记忆与主动关怀
- 移动端陪伴感

不要照搬：

- 本项目不是情感陪伴 App
- 不要早期引入复杂伦理和关系产品风险

## AI Dungeon

可借鉴：

- Context Viewer
- Plot Components
- Story Cards
- Memory System
- 导出备份意识

不要照搬：

- 不要过度游戏化
- 本项目更偏角色工作台

## NovelAI

可借鉴：

- Lorebook
- Memory
- Author's Note
- 写作编辑器式设定管理

不要照搬：

- 不要让 UI 太像纯写作编辑器
- 聊天体验仍然是核心

## SillyTavern

可借鉴：

- 角色卡
- World Info
- Group Chat
- Prompt Inspector
- 插件和高级控制

不要照搬：

- 默认界面不要太 power-user
- 高级能力放进 Studio/Pro Mode

## 综合吸收路线

- 学 Character.AI 的 Hall 和首屏可理解性。
- 学 AI Dungeon 的上下文可视化。
- 学 NovelAI 的设定工作台。
- 学 SillyTavern 的高级能力分层。
- 与 Replika 区分：这是角色扮演工作台，不是陪伴关系产品。
