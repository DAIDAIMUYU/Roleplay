- Provider Gateway 统一模型调用入口
- Mock Provider（Demo 访客专用）
- DeepSeek Provider
- OpenAI-compatible Provider 结构
- 用户 BYOK 临时保存 + 本设备保存
- API 连接测试
- Provider 错误标准化（401/429/超时/模型不存在）
- Settings → API Provider 页面的真实交互
