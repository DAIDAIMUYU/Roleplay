# 09_世界书_记忆_分支_回收站_备份

## 世界书

字段建议：

- title
- category
- triggers
- content
- priority
- enabled
- scope：global / character / session / persona
- max_turns
- token_estimate
- last_triggered_at
- trigger_count

功能：

- 关键词命中
- 优先级排序
- 最大注入条数
- token 预算限制
- 命中解释
- 手动强制注入
- 绑定角色/会话
- 导入导出

## 记忆

记忆类型：

- short_term
- long_term
- relationship
- event
- user_preference
- character_preference
- summary

第一版不建议自动无脑保存长期记忆。推荐：

```text
AI 建议记忆 → 用户确认 → 用户可编辑 → 保存
```

## 剧情摘要

先做 summary，再做复杂检索记忆。

- 手动生成摘要
- AI 建议摘要，用户确认
- 每 N 条消息提醒更新摘要
- 分支有独立摘要

## 分支剧情

字段：

- branch_id
- session_id
- from_message_id
- name
- created_at

功能：

- 从某条消息创建分支
- 分支重命名
- 分支切换
- 分支归档
- 分支导出

## 回收站

核心数据不直接物理删除：

- 角色
- 会话
- 消息
- 世界书
- 记忆
- 提示词模板

支持：

- 软删除
- 恢复
- 永久删除
- 清空回收站
- 操作审计

## 备份恢复

支持：

- 导出全部数据 JSON
- 导出角色
- 导出会话
- 导出世界书
- 导出提示词模板
- 导入前预览
- 冲突处理
- schema_version
- checksum

备份矩阵：

- 数据库
- Storage 对象
- Edge Functions / 环境配置
- Auth 设置
- 系统模板
