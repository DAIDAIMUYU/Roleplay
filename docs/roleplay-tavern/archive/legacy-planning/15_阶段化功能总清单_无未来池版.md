# 15_阶段化功能总清单：无未来池版

> V4.3 不再使用“未来功能池”作为模糊存放区。所有功能必须归属到明确阶段。若某能力不属于核心产品，则归入阶段 10，并标记为可选生态扩展。

## 功能优先级定义

| 优先级 | 含义 | 是否必须进入阶段 |
|---|---|---|
| P0 | 安全、数据、产品地基 | 必须，阶段 0-3 完成 |
| P1 | 核心聊天与 RP 能力 | 必须，阶段 4-6 完成 |
| P2 | 产品级完整体验 | 必须，阶段 7-9 完成 |
| P3 | 生态/商业化/多模态 | 可选，阶段 10 完成 |
