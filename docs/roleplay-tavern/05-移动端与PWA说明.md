# 移动端与 PWA 说明

## 1. 移动端网页使用

手机浏览器访问 Roleplay Tavern 仍属于网页模式，不是独立移动 App。

## 2. 底部导航

移动端底部导航固定在视口底部，用于切换首页、聊天、工坊和设置。页面内容需要为底部导航留出空间。

## 3. Modal / 弹窗行为

移动端打开 AppModal 时，底部导航会隐藏，避免遮挡保存按钮。长表单应在 Modal 内部滚动，footer 保持可见。

## 4. iOS 输入框缩放处理

移动端输入框字号保持不低于 16px，并锁定 viewport，减少 iOS Safari 聚焦输入框时的自动放大。

## 5. PWA manifest / Service Worker

项目保留 PWA manifest、Service Worker、更新提示和离线提示。

## 6. PWA 安装方式

页面内“安装角色酒馆”提示卡片已下线。用户仍可通过浏览器菜单或系统分享菜单添加到主屏幕。

## 7. 更新提示

如果检测到新版本，PWA 更新提示会引导刷新。若手机仍显示旧页面，可能是 Service Worker 或浏览器缓存导致。

## 8. 离线提示

离线状态下会显示提示。离线时无法调用真实 Provider，也可能无法访问云端同步能力。

## 9. 常见问题

### 手机看到旧版本怎么办？

可以尝试：

1. 等待部署完成后强刷。
2. 清理浏览器站点数据。
3. 删除主屏幕 PWA 图标后重新添加。
4. Chrome 可在 Application → Clear storage 清理。

### 底部按钮被导航遮挡怎么办？

请确认使用最新版本。如果仍发生，记录页面、设备、浏览器和复现步骤，提交到 GitHub Issues。
