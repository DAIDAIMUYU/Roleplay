# 用户使用指南

欢迎使用角色酒馆！本文档将帮助你从零开始使用各项功能。

---

## 一、访客 Demo 体验

打开应用后，即使不登录也可以点击 **Demo** 体验产品功能。

- Demo 模式使用 Mock AI，回复是模拟的
- 不会消耗任何 API 额度
- 数据不会保存到服务器

---

## 二、注册与登录

1. 点击右上角或页面中的 **登录** 按钮
2. 输入邮箱和密码
3. 首次使用请点击"注册"切换模式
4. 注册成功后自动登录

> 密码要求：至少 6 位字符

---

## 三、配置 API

登录后需要配置 AI Provider 才能开始真正的角色扮演。

### 方式一：本地 BYOK（推荐流式体验）

1. 进入 **设置** 页面
2. 在"保存模式"中选择 **仅本次会话** 或 **当前设备**
3. 选择 Provider（DeepSeek 或 OpenAI 兼容）
4. 填写 Model 名称和 Base URL（一般用默认值即可）
5. 输入你的 API Key
6. 点击 **保存配置**
7. 点击 **测试连接** 确认配置正确

- **仅本次会话**：关闭标签页后 API Key 自动清除
- **当前设备**：API Key 保存在浏览器中，下次打开仍可用

?> API Key 只保存在你的浏览器中，不会上传到服务器。

### 方式二：托管加密（推荐多设备用户）

1. 进入 **设置** 页面
2. 在"保存模式"中选择 **托管加密 / 跨设备同步**
3. 填写凭据名称、Provider、Model、API Key
4. 点击 **保存到托管加密**
5. 保存成功后 API Key 不会再显示明文

?> 托管加密模式通过服务端代理调用，当前为**非流式回复**（一次性返回完整内容）。如需逐字流式输出，请使用本地 BYOK 模式。

### 连接测试失败？

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| API Key 错误 (401) | Key 不正确 | 检查 Key 是否完整复制 |
| 额度不足 (402) | 账户余额不足 | 充值或更换 Key |
| 限流 (429) | 请求太频繁 | 稍等片刻再试 |
| 网络错误 | 无法连接 | 检查网络或 Base URL |

---

## 四、创建角色

1. 进入 **工作台 (Studio)** 页面
2. 点击 **新建角色**
3. 填写角色信息：
   - **名称**：角色的名字
   - **身份/设定**：角色的背景、性格描述
   - **外貌**：角色的外观描述
   - **开场白**：角色首次对话时的问候语
   - 其他字段可选填
4. 点击保存

> 角色卡支持收藏和归档。删除的角色会进入回收站，可以恢复。

---

## 五、创建会话

1. 进入 **聊天室** 页面
2. 点击左侧会话列表的 **+** 按钮
3. 选择：
   - **绑定角色**：选择一个已创建的角色开始角色扮演
   - **空白会话**：不绑定角色，自由聊天
4. 如果绑定了角色并有开场白，角色会自动发送首条消息

---

## 六、添加提示词模板

提示词模板用于在聊天中注入特定的对话规则或场景设定。

1. 进入 **工作台 → 提示词模板**
2. 点击 **新建模板**
3. 填写：
   - **名称**：便于识别的标题
   - **分类**：选择模板类型
   - **内容**：模板正文，可使用 `{{character_name}}` 等变量
4. 保存后，在聊天页右侧 **上下文控制台** 中点击添加按钮将模板加入当前会话
5. 模板可随时启用或禁用

---

## 七、添加世界书

世界书用于构建世界观设定，AI 会根据聊天内容自动匹配相关条目。

1. 进入 **工作台 → 世界书**
2. 创建一个世界书，然后添加条目：
   - **关键词**：触发词，逗号分隔（如"王都,城堡,皇家"）
   - **内容**：触发后注入 AI 的设定文本
   - **优先级**：数字越大越优先
   - **启用**：可随时开关
3. 在聊天页右侧控制台中将世界书添加到当前会话
4. 当聊天中出现关键词时，对应的世界书条目会自动注入 System Prompt

> 点击右侧的 Context Preview 可以看到哪些条目被命中（绿色）或跳过（红色）。

---

## 八、添加记忆

记忆用于保存重要的剧情信息或角色关系。

### 手动添加

1. 进入 **工作台 → 记忆**
2. 点击 **新建记忆**
3. 填写标题、内容、记忆类型
4. 保存后在聊天右侧控制台中添加到当前会话

### AI 提炼记忆

1. 在聊天页右侧控制台点击 **AI 提炼记忆**，或点击单条消息气泡上的"提炼记忆"按钮
2. AI 会分析最近对话并生成候选记忆
3. 在弹出的预览窗口中：
   - 编辑标题、内容、类型、重要性
   - 点击 **保存并启用** 直接激活记忆
   - 或点击 **保存到收件箱** 稍后确认
4. 处于"建议"状态的记忆不会注入 AI 上下文

---

## 九、生成摘要

对于长会话，可以自动生成摘要来控制上下文长度。

1. 在聊天页右侧控制台找到"会话摘要"区域
2. 点击 **AI 生成摘要**
3. 预览生成的摘要
4. 点击 **保存** 确认
5. 启用摘要后，AI 会在理解当前对话时参考这段摘要

---

## 十、编辑消息与版本切换

### 编辑消息

1. 将鼠标悬停在你的消息上（手机端点击消息右侧的 **⋯** 按钮）
2. 点击 **编辑** 按钮
3. 修改内容后点击 **重发**
4. AI 会基于新内容重新生成回复

### 查看历史版本

1. 消息编辑重发后，旧版本会自动保存
2. 消息气泡上会出现版本号（如"2 / 3"）
3. 点击左右箭头切换查看不同版本
4. 手机上版本箭头直接显示在气泡上，也可以从 **⋯** 菜单中切换

### 重新生成

- 对最新的 AI 回复可以点击 **重新生成**，让 AI 重新回答

---

## 十一、复制与删除消息

- **复制**：点击消息的复制按钮，将消息内容复制到剪贴板
- **删除**：点击删除按钮，消息会进入回收站（软删除），并非永久删除

---

## 十二、导出备份与导入恢复

### 导出备份

1. 进入 **设置 → 数据管理**
2. 点击 **导出 JSON**
3. 浏览器会下载一个 JSON 文件，包含：
   - 角色、提示词模板、世界书、记忆
   - 会话、消息、版本历史
4. 导出**不包含** API Key 和托管凭据

### 导入恢复

1. 进入 **设置 → 数据管理**
2. 点击 **导入 JSON**
3. 选择之前导出的备份文件
4. 预览导入内容
5. 选择导入模式：
   - **作为副本导入**（推荐）：生成新 ID，不覆盖现有数据
   - 可选 **跳过已存在项目**
6. 点击 **开始导入**

### 回收站

1. 进入 **设置 → 数据管理 → 回收站**
2. 可以看到所有已删除的项目
3. 支持恢复：会话、消息、角色、模板、世界书、世界书条目、记忆
4. 恢复世界书时可以选择同时恢复其条目

---

## 十三、移动端使用

手机浏览器访问时会自动适配移动端布局：

- 底部导航：聊天 / 工作台 / 设置
- 左侧会话列表：点击左上角按钮展开
- 右侧上下文控制台：点击右上角按钮展开
- 消息操作：点击消息气泡上的 **⋯** 按钮弹出操作菜单
- 版本切换：直接显示在消息气泡上，点击箭头即可

---

## 十四、快捷操作总结

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 发送消息 | Enter | 发送按钮 |
| 换行 | Shift + Enter | 换行键 |
| 停止生成 | 点击停止按钮 | 点击停止按钮 |
| 编辑消息 | hover → 编辑 | ⋯ → 编辑 |
| 复制消息 | hover → 复制 | ⋯ → 复制 |
| 删除消息 | hover → 删除 | ⋯ → 删除 |
| 版本切换 | hover → 箭头 | 点击箭头 或 ⋯ → 版本 |
| 提炼记忆 | hover → 大脑图标 | ⋯ → 提炼记忆 |
| 重新生成 | hover → 旋转图标 | ⋯ → 重新生成 |
