# 01_产品定位与商业目标

## 一句话定位

角色酒馆是一个面向长期角色扮演的私人 AI 角色工作台，同时提供一个可公开展示的 Demo 酒馆大厅。

## 主定位

面向创作者和长期角色扮演用户，提供：

- 角色卡管理
- 会话与剧情管理
- 世界书与记忆管理
- 上下文透明化
- 多 Provider BYOK
- 移动端 App 化体验
- 数据备份、恢复、导入导出

## 副定位

对外展示一个“看得懂、能试玩、不烧钱、不泄露数据”的 Demo 大厅。访客只看演示，不进入真实用户权限面。

## 明确非目标

早期不做：

- 开放社区平台
- 大规模角色市场
- 陪伴关系产品
- 托管式多租户 SaaS
- 访客免费使用站主 API
- 复杂支付系统
- 团队协作空间

## 北极星指标

早期：

- 访客 Demo 完成率
- 注册转化率
- BYOK 首次连接成功率
- 首次真实聊天成功率
- 移动端聊天可用性

中期：

- 周留存
- 角色卡创建数
- 会话持续轮次
- 上下文预览使用率
- 导出备份使用率

后期：

- 公开模板/分享使用量
- PWA 安装率
- 高级功能激活率
- 托管 Provider 转化率，如决定做

## 商业化路线建议

短期不急着收费。先验证：

1. Demo 能不能让人理解产品。
2. 登录后 BYOK 能不能顺利跑起来。
3. 用户愿不愿意把角色、世界书、剧情长期留在站内。

中后期再考虑：

- 高级备份与同步
- 托管 Provider 额度包
- 私有部署服务
- 模板/角色市场
- 高级上下文调试器
- 多设备 PWA/APP 体验增强
