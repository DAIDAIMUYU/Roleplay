# Roleplay Feature Module

本目录为角色扮演核心功能模块，按阶段逐步构建：

- **components/** — UI 组件
- **pages/** — 页面级组件
- **hooks/** — 自定义 hooks
- **services/** — 业务逻辑层
- **repositories/** — 数据访问层
- **providers/** — Provider 与 API 调用
- **storage/** — 本地存储
- **types/** — 类型定义
- **utils/** — 工具函数
- **mock/** — Mock 数据与 Demo 回复

阶段 0 仅建立目录结构，后续阶段逐步填充。
